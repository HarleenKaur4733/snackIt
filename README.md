# 🍔 Full Stack Food Ordering Application

A modern full-stack Food Ordering Web Application that allows users to browse food items, place orders securely, manage carts, and track orders, while providing administrators with a powerful dashboard to manage the platform efficiently.

---

# 🚀 Features

## 👤 User Features

* User Registration & Login Authentication
* Secure JWT-based Authentication
* Browse Food Categories & Menu Items
* Search and Filter Food Items
* Add to Cart Functionality
* Quantity Management in Cart
* Online Payment Integration using Stripe
* Place Orders Securely
* Order History & Tracking
* Responsive UI for Mobile and Desktop

---

## 🛠️ Admin Features

* Admin Authentication & Authorization
* Admin Dashboard with Analytics
* Add / Update / Delete Food Items
* Manage Categories
* View All Orders
* Update Order Status
* Manage Users
* Revenue & Order Statistics
* Secure Admin Routes

---

# 🔒 Security Features

* JWT Token Authentication
* Protected Routes
* Role-Based Access Control (Admin/User)
* Password Encryption
* Secure Payment Handling with Stripe
* API Validation
* Error Handling & Exception Management
* Environment Variable Protection using `.env`
